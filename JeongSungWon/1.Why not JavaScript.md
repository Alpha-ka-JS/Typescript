### Typescript의 사용 이유
**타입의 안정성을 위해**

1. 말도 안되는 코드
```JS
$ [1, 2, 3] + false
```
위의 코드를 실행하면 다음과 같은 결과값이 나온다.

```
'1,2,3false'
```

2. 잘못된 인자가 들어간 경우
```JS
const divide = (a, b) => a / b;
```
다음과 같은 divide를 선언한 경우
```
$ divide('xxx')
```
위에 같이 입력을 하게되면 **NaN** 이 나온다.

3. 잘못된 코드인데 컴파일 시킴
```JS
const info = { name:"sungwon" }
```
위의 코드를 선언하고
```
$ info.hello()
````
를 실행하게 되면 잘못된 코드지만 실행하게 된다.

위와 같은 이유로 Typescript를 사용하게 된다.
